============================================================
Project: Real-Time Face and Smile Detection using Haar Cascades
============================================================

This script implements a real-time computer vision application that 
detects human faces and smiles using a webcam feed. It utilizes 
OpenCV's pre-trained Haar Cascade classifiers to identify features 
hierarchically.

---

### 1. Cascade Classifiers Initialization
The script begins by loading three XML files containing pre-trained 
knowledge for detecting specific patterns:
- face_cascade: Detects the general structure of a human face.
- eye_cascade: Detects eyes (loaded but optional in this logic).
- smile_cascade: Specifically trained to detect the mouth area when smiling.

---

### 2. The Detection Logic (detect function)
This function processes individual frames to find and mark features:

A. Face Detection:
   - Uses 'detectMultiScale' on the grayscale image. 
   - A scale factor of 1.3 and a neighbor threshold of 5 are used to 
     balance speed and accuracy.
   - Draws a Blue rectangle (255, 0, 0) around any detected face.

B. Region of Interest (ROI) Extraction:
   - To save processing power and reduce false positives, the script 
     defines an "ROI." It only looks for smiles INSIDE the area where 
     a face has already been found.
   - roi_gray: Used for the mathematical detection.
   - roi_color: Used for drawing the final visual rectangle.

C. Smile Detection:
   - Uses 'smile_cascade' within the face ROI.
   - Parameters (1.8, 20): A higher neighbor threshold (20) is used 
     to ensure the smile is clear, preventing the code from 
     detecting a normal mouth as a smile.
   - Draws a Red rectangle (0, 0, 255) around the detected smile.

---

### 3. Main Execution Loop

1. Video Capture: 
   - Opens the default system camera using cv2.VideoCapture(0).

2. Frame Processing:
   - Converts every frame to grayscale (cv2.COLOR_BGR2GRAY) because 
     Haar Cascades do not require color information to detect shapes.
   - Passes the grayscale and original color frames to the detect() function.

3. Visualization:
   - Displays the live feed in a window named 'Video' with the 
     bounding boxes overlaid on the user's face.

4. Exit Strategy:
   - The loop runs continuously until the 'q' key is pressed.
   - Properly releases the hardware (webcam) and closes GUI windows 
     upon exit to prevent system lag.

---

### 4. Requirements
- Python 3.x
- OpenCV (opencv-python)
- Pre-installed Haar Cascade XML files (included with OpenCV)
